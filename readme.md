<strong>A simple cross platform pomodoro timer
</strong>
## How to run the app locally

#### Clone & install
```
git clone https://github.com/dilpreetsio/Pomododo.git
npm i
```

#### Run the app

```
npm start
```

#### Build executables
```
npm run build
```
